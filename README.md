# STMOVビュアー v1.1.0

Sports Time Machine（スポーツタイムマシン）の3D点群動画ファイル（STMOV形式）を再生するWebアプリケーションです。

## クイックスタート

```bash
# 1. 環境設定ファイルをコピー
cp .env.example .env

# 2. インストール（初回のみ）
npm install

# 3. 起動
npm start

# 4. ブラウザで開く
# http://localhost:3001
```

> Dockerで使ってみたい場合は[使い方](./使い方.md)へ
